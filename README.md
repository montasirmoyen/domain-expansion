# domain-expansion

This program uses a webcam to detect specific hand gestures and identifies which Domain Expansion hand sign they correspond to. When a gesture is recognized, the program displays the name of the Domain Expansion associated with that hand sign.

The goal is to demonstrate real-time hand tracking and gesture recognition using computer vision.

### Backstory

A "Domain Expansion" (領域展開, りょういきてんかい, Ryōiki Tenkai) is an advanced barrier technique and is considered the pinnacle of jujutsu sorcery in the popular Japanese dark fantasy manga and anime series created by Gege Akutami [(Wiki)](https://jujutsu-kaisen.fandom.com/wiki/Domain_Expansion).

# Usage

```
git clone https://github.com/montasirmoyen/domain-expansion
cd domain-expansion

pip install -r requirements.txt

python3 src/main.py
```

Pressing "Q" on your keyboard closes the program.
